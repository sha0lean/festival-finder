## Sources générales

| Source | Usage | Fiabilité |
|---|---|---|
| Site officiel festival | dates, tickets, camping, line-up | 🟢 Maximum |
| Billetterie officielle | prix, phases, sold-out | 🟢 Fort (Dice, Shotgun, Ticket Fairy, Cooltix, Eventbrite) |
| Resident Advisor | clubs et events électroniques | 🟢 Fort |
| Instagram / Facebook officiel | annonces tardives open-airs | 🟡 Moyen (vérifier l'organisateur) |
| Songkick / Bandsintown | tournées artistes et labels | 🟡 Moyen — utile DnB/dub |
| Google Maps | GPS exact, parking, accès baignade | 🟢 Fort pour localisation |
| OpenStreetMap | environnement, lac, forêt | 🟢 Fort pour cadre nature |

### Fiabilité par type de source

| Niveau | Source | Autoriser pour carte ? |
|---|---|---|
| 🟢 Fort | Site officiel / billetterie officielle | Oui |
| 🟢 Fort | Post officiel organisateur (avec lien) | Oui |
| 🟡 Moyen | Resident Advisor / Shotgun / Dice | Oui si event actif |
| 🟡 Moyen | Article média spécialisé | Complément seulement |
| 🔴 Faible | Blog / vieux forum / Reddit | Non pour dates et prix |
| 🔴 Faible | Souvenir édition précédente sans source | Non — estimation seulement |
